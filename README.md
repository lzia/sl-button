sl-button
============

See the [component page](http://lzia.github.io/sl-button) for more information.

## Getting Started

### Styling
```css
  sl-button.foo {
    --radius: 100px;
    --bgcolor: blue;
    --font-size: 30px;
    --font-color: white;
    --ripple-color: white;
    --font: 'Helvetica';
  }
```
### Using
```html
  <sl-button class="foo" label="X" action="fire"></sl-button>
```
## Browser Support
Currently only Firefox (Shame on you, Chrome).
